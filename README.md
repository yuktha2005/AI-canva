# Air Canvas using MediaPipe & OpenCV



Draw in the air using your hand! This project uses **MediaPipe** for hand tracking and **OpenCV** for drawing on a virtual canvas. You can draw with your index finger, select colors using THUMB , and erase the canvas in real-time.

## Features

- Draw in real-time using your index finger.
- Select colors using thumb.
- Erase using the "ERASER" option.
- Clear the canvas with the 'C' key.
- Quit the application with the 'Q' key.

## Requirements

- Python 3
- OpenCV
- MediaPipe
- NumPy
